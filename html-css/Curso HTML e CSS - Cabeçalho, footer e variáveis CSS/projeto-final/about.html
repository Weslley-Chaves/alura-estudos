Sobre mim

Sou Weslley Chaves, recém-formado em Análise e Desenvolvimento de Sistemas e em transição de carreira para a área de desenvolvimento de software e ciência de dados.

Minha trajetória começou na área administrativa, onde atuei com faturamento e processos financeiros. Essa experiência me deu organização, atenção aos detalhes e visão de processos — habilidades que hoje aplico na tecnologia.

Durante a faculdade, trabalhei com Análise de Requisitos, mapeando demandas e documentando soluções. Essa vivência me mostrou o impacto do software e reforçou minha vontade de seguir para o desenvolvimento e dados. Atualmente, estudo em tempo integral, com foco prático em:

  • Python & SQL – automação, análise e manipulação de dados
  • HTML, CSS e JavaScript – fundamentos de front-end
  • Git & GitHub – versionamento e automação

Sou movido por aprendizado contínuo, gosto de desafios e busco minha primeira oportunidade como desenvolvedor back-end ou analista/cientista de dados júnior.
